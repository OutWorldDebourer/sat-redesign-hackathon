# SAT Lima Redesign Hackathon

Prototipo frontend local para reestructurar la experiencia informativa del SAT Lima con React, Vite y TypeScript.

## Objetivo

Crear una web civica, clara y accionable para orientar a ciudadanos en consultas, pagos, papeletas, tributos, fraccionamiento, sedes y tramites digitales. El prototipo incluye un asistente SAT persistente para guiar rutas ciudadanas.

## Scripts

```bash
npm install
npm run dev
npm run build
```

## Stack

- React
- Vite
- TypeScript
- React Router
- Lucide React

## Nota

Este prototipo no conecta APIs reales. Las interacciones y respuestas del asistente son simuladas para demo de hackathon.
