# 07. Programas y recursos

## Programas

### Contribuyente puntual

Fuentes:

- https://www.sat.gob.pe/websitev9/Programas/ContribuyentePuntual
- https://www.sat.gob.pe/websitev9/Servicios/ContribuyentePuntual

Programa orientado a reconocer o beneficiar a contribuyentes que cumplen oportunamente sus obligaciones.

Uso para rediseño:

- Explicar requisitos.
- Mostrar beneficios.
- Permitir verificar condicion.
- Conectar con pagos y vencimientos.

### Cultura tributaria

Fuente:

- https://www.sat.gob.pe/websitev9/culturatributaria

Seccion educativa para promover conocimiento y cumplimiento tributario.

Uso para rediseño:

- Convertir en centro de aprendizaje.
- Segmentar por ciudadanos, colegios, empresas y nuevos contribuyentes.

### Info Movil SAT

Fuente:

- https://www.sat.gob.pe/websitev9/Servicios/InfoMovilSAT

Servicio o programa movil del SAT. Debe validarse detalle operativo antes de diseñar componentes especificos.

### Responsabilidad social

Fuente:

- https://www.sat.gob.pe/websitev9/responsabilidadsocial

Seccion institucional asociada a acciones de responsabilidad social.
